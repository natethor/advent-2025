# Advent of Code 2025 - Zig

Learning Zig through Advent of Code challenges.

## Running Solutions

Run any day directly:

```bash
zig run src/day01.zig
```

With optimizations (faster execution):

```bash
zig run -O ReleaseFast src/day01.zig
```
